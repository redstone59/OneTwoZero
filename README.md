# One Two Zero
A program that links to OBS's websocket API to cause allow for certain events to happen when certain criteria are met.
Created for the *One Two Zero Stream project*, where the stream automatically ends after 2 hours.

# Requirements
- OBS Websockets v5 (included with OBS)
- The [python_obs](https://github.com/redstone59/Python-OBS) module.
- The [Advanced Scene Switcher plugin](https://obsproject.com/forum/resources/advanced-scene-switcher.395/)

>[!important]
>This repository uses a [fork of SamChichester/Python-OBS](https://github.com/redstone59/Python-OBS) to allow for support for Advanced Scene Switcher's websocket macro activation.
